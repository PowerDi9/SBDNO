# SBDNO - Small Business Delivery Note Organizer

## Description

SBDNO (Small Business Delivery Note Organizer) is a desktop application developed in Java with a Swing graphical interface, designed to organize and manage delivery notes for small businesses. The application provides tools to efficiently create, edit, search, and manage delivery notes. You can also generate listings and daily reports for the delivery notes stored on the database, on .xlsx format.

## Key Features

- Complete management of delivery notes
- Intuitive graphical interface developed with Swing
- Search and filter delivery notes by various criteria
- Report and listing generation
- Local data storage
- Design optimized for Windows devices

## System Requirements

- **Operating System:** Windows 10 or higher (backward compatible)
- **Java Runtime Environment:** Version 17 or higher
- **RAM:** Minimum 2 GB recommended
- **Disk Space:** Minimum 100 MB for application and data

## Installation

### Prerequisites

1. Verify that Java 17 or higher is installed on the system:
   ```
   java -version
   ```

2. If Java is not installed, download and install from:
   - [Oracle Java JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)
   - Or use an OpenJDK 17 distribution

### Application Installation

1. Download the SBDNO installation file
2. Run the installer and follow the instructions
3. The application will install to the selected location
4. A shortcut will be created in the start menu and/or desktop

## Usage

1. Launch the application from the shortcut or start menu
2. Use the main menu to access different functionalities:
   - Create new delivery notes
   - View existing delivery notes
   - Search delivery notes by specific criteria
   - Generate reports

## Development

### Development Requirements

- JDK 17 or higher
- Java-compatible IDE (Eclipse, IntelliJ IDEA, NetBeans)

### Compilation from Source Code

1. Clone or download the repository
2. Import the project into your preferred IDE
3. Configure JDK 17 as the execution environment
4. Compile and run the application from the main class

## Database

SBDNO uses an embedded database (SQLite) to store information locally.
